Clover - Solusi Tabbed File Explorer

Jan 05, 2019 in productivity

Pengguna linux pasti ga asing dengan File Manager yang ada tab nya. Iya kan? Pasti susah move on dari itu, jujur aja… haha. Kalau di windows, File Explorer tidak ada fitur tab. Jadi kalau kita mau buka lebih dari satu File Explorer, salah satu caranya ya buka File Explorer baru. Yang terjadi adalah, desktop kita berantakan, hahaha. Di Windows, ada beberapa aplikasi 3rd-party untuk fitur tabbed explorer, salah satu yang penulis suka yaitu Clover

Clover? apasih?

Clover merupakan 3rd-party aplikasi yang sama dengan File Explorer, tapi dengan tambahan fitur multi-tab nya. Nih contoh tampilan aplikasinya,

Jadi kita bisa buka banyak File Explorer, tapi masih dalam satu window, jadi kan ga berantakan.

Keren nih, cara installnya?

Langsung download aja kesitu. “Bahasanya aseng bos, ga suka gua”, yaudah ga usah install. Bagi yang ga rasis, nih tak kasih tau rekomendasi, jangan centang yang kanan bawah.

Kalau udah, tinggal teken aja tombol gede di tengah itu. Udah gitu sih. Sekian Terima Kasih
