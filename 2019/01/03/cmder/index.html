Cmder? Commander? Command Prompt er? Apasih?

Jan 03, 2019 in productivity

Cmder itu ya semacam command prompt, tapi dengan fitur yang lebih banyak. Kalau kalian pernah menggunakan Linux atau Mac OS, pasti kenal yang namanya terminal kan? Kalau kalian bandingin dengan Command Prompt nya windows, bagusan mana? IMO sih bagusan linux jauh!!

Maka dari itu, Cmder solusi buat kalian yang boring banget dengan Command Prompt nya Windows!!

Fiturnya apasih?

Kalau fitur saya ga tau lengkapnya, cuman yang paling saya sukai yaitu fitur multitab sama context menu. Kalau mau tau lengkapnya, baca aja disini Cmder.

Nih tak kasih skrinsut

Keren oi! Cara masangnya gimana?

Nih, gampang banget. Kayak install aplikasi biasa, huehuehue,

  1. download aja disini
  2. Unpack ke folder yang kamu suka, kalau saya sih aplikasi standalone / portable gini biasa tak taruh di C:\Apps
  3. Buat aktifin Context Menu yang klik kanan - cmder here, buka aja Cmder.exe pake Administrator
  1. Ketik .\cmder.exe /REGISTER ALL terus enter. Beres
  2. Kalau keluar pesan “Access Denied”, berarti cmder mu belum admin. Balik ke langkah 3
  3. Udah klik kanan di Windows Explorer, nanti juga muncul cmder here.

Sekian dan terima kasih

OIT! ada 1 masalah.

Kalau kamu bukan seorang pengguna rutin cmd, ini tidak begitu masalah. Jadi abaikan pesan ini. Tapi yang mau tau, silahkan baca, masalahnya yaitu, ketika kalian tambahkan Environment Variable, cmder tidak bisa update!! Walaupun udah kalian close, ga bakal bisa update. Solusinya ya restart komputer. Udah gitu sih.
