WoX, alternatif untuk aplikasi Alfred dan Launchy

Jan 03, 2019 in productivity

WoX itu simplenya ya search gitu, kalau kalian pencet tombol windows, terus ngetik nama aplikasi. Nah kayak gitu WoX itu! “Lah kan? ngapain pake WoX, kan sama fungsinya.”, biar keren aja getoh. Liat, keren kan?

Untuk lebih lengkapnya, bisa dilihat disini

Cara makenya gimana?

Baca sih di githubnya, kan lengkap. “Aku ga paham bahasa inggris mas.”, ouh….. oke tak bahas deh.

  1. Download dulu WoX nya disitu
  2. Sebenernya tinggal run aja udah bisa kok, cuman katanya di github, syaratnya harus install ini everything sama python3. Ikutin aja, biar ga ada error, huehuehue.
  3. Kalau udah, buat nampilin dan nyembunyiin WoX nya pake “alt+space”

Fitur

  1. Kalian bisa searching apapun pake WoX, entah itu aplikasi, folder, file, dsb.
  2. Pake keyword pinyin buat nyari program (cuman translate doang, ga pernah pake hehehe)
  3. Searching di google tinggal tulis didepannya “g”, misal g WoX.
  4. Bisa ganti tema WoX
  5. Ada banyak plugin, bisa dicari disini

Sekian dan terima kasih
